Description

  • Saber is a single line message bubble theme for Messages.app.

    Saber requires MessagesCustomizer which can be found on chewitt.me/repo. In the tweak settings, turn on "Bubble Outlines Only".

    Originally inspired by /u/NickBarrow's message bubbles.
